
# Calculator

## 📌 Project Overview
This Calculator is a simple and responsive web application developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations with an easy-to-use interface.

## ✨ Features
- Addition, Subtraction, Multiplication, and Division
- Clear (AC) button
- Responsive design
- User-friendly interface
- Real-time calculations

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
