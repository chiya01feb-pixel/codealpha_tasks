# codealpha_tasks
# Multi-Project Repository

This repository contains multiple projects, each maintained in its own dedicated Git branch.

## Available Branches

### 🧮 `calculator`

Contains the Calculator application and all related source code.

### 🎵 `music-player`

Contains the Music Player application and all related source code.

## How to Access a Project

Clone the repository and switch to the branch corresponding to the project you want to work with.

```bash
git clone <repository-url>
cd <repository-name>

# Calculator Project
git checkout calculator

# Music Player Project
git checkout music-player
```

## Branch Structure

| Branch         | Project                                  |
| -------------- | ---------------------------------------- |
| `main`         | Repository information and documentation |
| `calculator`   | Calculator application                   |
| `music-player` | Music Player application                 |

The `main` branch is used only for repository documentation. Each project is developed and maintained independently in its respective branch.
