# 🎵 Music Player

A modern and responsive web-based Music Player developed using **HTML, CSS, and JavaScript**. This application provides an intuitive interface for playing audio tracks with essential playback controls and a seamless user experience.

## 📖 Overview

The Music Player is designed to demonstrate core front-end development concepts, including DOM manipulation, event handling, and the HTML5 Audio API. It enables users to play, pause, and navigate through multiple audio tracks with ease.

## ✨ Features

- Play and pause audio
- Next and previous track navigation
- Multiple song playlist support
- Responsive and user-friendly interface
- Clean and interactive design
- HTML5 Audio API integration

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

Music Player/

## 🚀 Getting Started

1. Clone or download the repository.
2. Open the **Music Player** folder.
3. Launch `index.html` in any modern web browser.
4. Use the playback controls to enjoy your favorite tracks.
